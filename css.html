CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed
A CSS rule consists of a selector and a declaration block:


.
The selector points to the HTML element to style (h1).
The declaration block (in curly braces) contains one or more declarations separated by semicolons. Each declaration includes a CSS property name and a value, separated by a colon.
